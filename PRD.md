# Vocab Master — PRD

> 词汇大师 · 多教材英语词汇学习系统  
> 版本 2.0 · 2026年7月

---

## 1. 产品概述

### 1.1 定位
基于间隔重复记忆法的多教材英语词汇学习工具，支持 PET/KET/四级/雅思等多套教材自由切换，Web 和 iOS 双端运行。

### 1.2 目标用户
- 备考剑桥英语（PET/KET）的中学生
- 备考大学英语四六级的大学生
- 需要系统积累词汇的英语学习者

### 1.3 核心价值
- **科学记忆**：Leitner 间隔重复算法，在遗忘临界点提醒复习
- **多教材自由切换**：一套系统覆盖多种考试，进度独立
- **全平台同步**：Supabase 云端存储，Web/iOS 实时同步

### 3.4 间隔复习 🧠

**算法：Leitner 间隔重复系统**

| Level | 间隔 | 说明 |
|-------|------|------|
| 0 | 1 天 | 刚标记"不认识" |
| 1 | 3 天 | 第一次认识 |
| 2 | 7 天 | 短期巩固 |
| 3 | 14 天 | 中期记忆 |
| 4 | 30 天 | 长期记忆 |
| 5 | 60 天 | 稳定掌握 |
| 6 | 90 天 | 毕业（不再主动复习）|

**规则：**
- 不认识 → 进入系统（Level 0，明天复习）
- 认识 → 升级（间隔翻倍）
- 再次不认识 → 重置（Level 0）
- Level 6 → 毕业

**入口：**
- 首页统计行 "🧠 今日复习 N 词"
- 首页 mode-grid "🧠 间隔复习"
- 导航栏 "🧠 复习"
- 闪卡结束页 "去复习 →"

**联动：**
- 闪卡/测验/拼写 → 不认识自动进入复习系统
- 错词页"已掌握" → 复习升级

## 4. 数据存储

### 4.1 localStorage

| Key | 内容 |
|-----|------|
| `active_textbook` | 当前教材 ID |
| `pvm_progress_{textbookId}` | 教材进度 JSON |
| `pvm_tts_voice` | TTS 语音偏好 |
| `pvm_cached_{textbookId}` | 远程教材缓存（仅非内置）|

**进度 JSON 结构：**
```json
{
  "wordStates": { "ability": "known", "abandon": "learning" },
  "wrongWords": ["abandon"],
  "starredWords": ["action"],
  "reviewSchedule": {
    "abandon": {
      "level": 2,
      "nextReview": "2026-07-22",
      "history": [
        { "date": "2026-07-18", "result": "wrong" },
        { "date": "2026-07-19", "result": "correct" }
      ]
    }
  },
  "game": {
    "xp": 320, "streak": 5, "level": 4,
    "hearts": 5, "todayXP": 45,
    "todayDate": "2026-07-19",
    "achievements": { "first10": true, "streak7": true }
  }
}
```

### 4.2 Supabase `user_progress` 表

| 列 | 类型 | 说明 |
|----|------|------|
| `user_id` | UUID (PK) | 用户 ID |
| `active_textbook` | TEXT | 当前教材 ID |
| `textbook_progress` | JSONB | 所有教材的进度聚合 |
| `custom_words` | JSONB | 用户额外导入的词（已禁用）|
| `updated_at` | TIMESTAMP | 最后更新时间 |

**`textbook_progress` JSONB 结构：**
```json
{
  "pet2020": {
    "word_states": { ... },
    "wrong_words": [...],
    "starred_words": [...],
    "review_schedule": { ... },
    "game_data": { ... }
  }
}
```

## 5. 性能优化

| 优化项 | 方式 |
|--------|------|
| 搜索防抖 | 250ms |
| 词表渲染 | 批量 innerHTML（一次写入）|
| UNIT 缓存 | `unitMap` 预计算，O(1) 查找 |
| CSS transition | 仅过渡指定属性 |
| 配对计时器 | requestAnimationFrame 替代 setInterval |
| 渲染按需 | renderAll 只渲染可见页面 |
